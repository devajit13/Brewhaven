# ☕ Brewhaven Coffee Shop

A modern, responsive coffee shop website built using **HTML, CSS, and JavaScript**. The website features a premium design, smooth animations, and multiple pages to showcase the coffee shop's menu, gallery, and contact information.

## 🚀 Features

- Responsive design for all devices
- Modern Hero Section
- Featured Coffee Section
- Popular Menu
- About Us
- Gallery
- Customer Testimonials
- Why Choose Us
- Newsletter Subscription
- Contact Page
- Mobile Navigation Menu
- Smooth scrolling and animations
- Clean and well-organized code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts

## 💻 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/coffee-shop.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or build tools are required.
